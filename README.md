# 📚 Library Management System

## 🚀 Overview
This project is a Library Management System developed using Java. It helps manage books, users, and borrowing operations efficiently through a structured system.

## 🔥 Features
- Add, update, and delete books
- Issue and return books
- Track available and issued books
- Manage user records
- Basic data handling and validation

## 🛠️ Tech Stack
- Java
- OOP (Object-Oriented Programming)

## ⚙️ Functionality
- Book Management Module
- User Management Module
- Issue/Return System

## 📂 Project Structure
src/
 ├── model
 ├── service
 ├── main

## ▶️ How to Run
1. Clone the repository
2. Open project in IDE (IntelliJ / Eclipse)
3. Run main class
